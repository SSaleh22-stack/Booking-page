# Public Assets

## Logo Setup

To add the Qassim University logo:

1. Place the logo file in this directory as `logo.png` or `logo.svg`
2. Recommended dimensions: 64x64 pixels or larger (will be resized automatically)
3. Supported formats: PNG, SVG, JPG, WebP

The logo will automatically appear in the header. If no logo is provided, a placeholder with "QU" will be shown.
